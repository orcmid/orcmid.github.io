Convergence of Document Management, Records Management, and the Web:

 Standards to the Rescue

version 0.90 last updated 2002-08-10-16:14 -0700 (pdt)

1. Seminar: April 23, 2002

download Microsoft PowerPoint slides

The seminar presentation illustrated ways business and government around the world are  using Web standards to improve coordination, communication, and services using the Web and XML standards.  There is demand for interoperability of systems in new ways.  Increase in interoperability requires more standards for metadata agreement and document interchange/preservation.

The new standards and activities are bringing document management and record management together as part of electronic content management.  Public accountability for eGovernment and citizen services makes management and preservation of electronic documents very important.  There is a general expectation that the Web and XML will make delivery of services and managing of records less expensive.

WebDAV is an example of new standards for inexpensive storage of managed documents.  So far, WebDAV standards do not provide for full integration of document management and records management in work activities.  This gap is a challenge to document management vendors.  This gap is also opportunity for document-management company and integrators to contribute valuable service in making standards useful.

4.1 WebDAV Activity

www.webdav.org
Access to all materials, projects, and announcements about WebDAV.
 
ietf.org
The 54th IETF meeting is being held in Yokohama, Japan, on 14-19 July 2002.
There is expected to be activity on a new draft of the ACL proposal and on advancing rfc2518 to Draft Standard status
 
draft-ietf-webdav-acl-08.txt (2002 May 7, 137kB text file)
This is draft 08 now being circulated for review, pending availability at the IETF site.  It is also available for viewing as HTML and viewing/downloading as a PDF.  This draft is in final review for agreement and discussion at IETF 54 in Yokohama, Japan, July 14-19, 2002.  There is a supplemental draft-ietf-webdav-acl-08.1 (2002 June 5) which reflects the one substantive change being made to draft -08 so far.
 
draft-reschke-webdav-search-01 (2002 April 9 or later working draft, on-line)
This is a working draft being developed as a new DASL proposal.  It is being discussed on the DASL discussion list.  There is no longer an official DASL Working Group.  Some people believe that the work on XML Query will make DASL unnecessary.  This has seemed to be a misunderstanding in the past.  The latest information on XML Query was posted on 2002 April 30.

4.2 WebDAV Implementation Information

Sharemation Server
The Xythos WebDAV implementation was the first WebDAV-specific implementation of a server.  The AIIM 2002 WebDAV Tutorial relied used a Sharemation account for examples.  
 
mod_dav_dasl (223k PDF file for download)
A specification for DASL additions to mod_dav, the implementation of WebDAV for the Apache web server.
 
DAV4J
This is an IBM Java implementation of classes for accessing and delivering WebDAV functions.  The documentation can be downloaded separately, after agreeing to a license.  There is also an open-source implementation that can be accessed on IBM DeveloperWorks.  Source code access requires use of CVS for public download.  The API Overview can be viewed on-line.
   This development is interesting because of its provision of a "wrapper" that encapsulates the WebDAV protocol and that is potentially adaptable to integration of other protocols for access to managed and versioned documents.
