ODMref1.0
     ODMA 1.0 Reference Implementation

ODMref1.0 is a miniature document-management system for personal use on desktop computers.  It is useful for demonstration, study,  training, troubleshooting, and prototyping of managed-document applications.
    

Features

  

System Requirements
  • Straightforward Desktop Operation
    Manage documents directly from ODMA-aware productivity applications; preserve and access your files in their original format with easy import, export, and packaging for interchange.
        
  • Simple Installation and Maintenance
    Software installation, configuration, backup, and removal preserves document collections across upgrades, downgrades, and re-installation.
      
  • Automatic Log and Audit Functions
    Activity logs provide an audit trail and complete details for verifying office-document operations involving ODMref1.0-managed documents.
        
  • ODMA 1.0 Conformance
    The ODMA 1.0 interface common to all ODMA-aware desktop software is implemented.
        
  • Full Software Source-Code and Documentation
    ODMref1.0 software and documentation illustrate, explain, and demonstrate the basic model for making document-management systems ODMA-compliant.  The software is freely-available for re-use and modification under an easy-to-honor open-source license.
        
  • Computer
    Microsoft® Windows®-based personal computer suitable for office-document desktop applications
        
  • Operating System
    Microsoft® Windows® XP Professional SP2 and Vista
        
  • Office Software
    ODMA-aware desktop applications such as Microsoft® Office Word 2003 and Office System Word 2007
        
  • Hard Disk
    50 MB available hard-disk space for initial installation and use (NTFS required)
        
  • Document Storage
    Local hard-disk space for keeping checked-in documents
     
  • Optional Developer Resources
    Working with ODMref1.0 source code requires Microsoft® Visual C++® and other freely-available tools
