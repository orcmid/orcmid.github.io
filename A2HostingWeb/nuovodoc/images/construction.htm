Construction Log

This images directory is part of the typical web structure for the NuovoDoc.com site and its mirrors.  The directory is used to carry images that are used at the overall NuovoDoc level.  The directory has a construction-oriented index.htm page and shares the common construction organization of the NuovoDoc site.

Change History

2002-01-26-14:50 Create NuovoDoc Image directory (orcmid)
An image directory under NuovoDoc is created so that NuovoDoc serves as the anchor of a mirror group.  (There are no peer members at this point, though there are certainly some one-way projects that would work.  A one-way project is one that points to NuovoDoc as its anchor but that isn't necessarily linked from NuovoDoc except in an exceedingly-careful way.  This might be sort of half-way loosely-coupled.)
1. The /image directory of the infonuovo.com anchor is copied here.  The only images that are kept are hardhat-logo.gif and NuovoDoc-logo.gif (renaming of nuo010405-logo.gif).
2. The only pages that are kept are construction.htm and index.htm.  nfoCentrale.htm style requirements are already satisfied.
3. This Construction Log page is edited to reflect its role and correct history, treating the original as boilerplate.  The correct images are connected and the construction-history block is adjusted.
4. The index.htm page is edited to be specific to NuovoDoc /images.
5. The term "NuovoDoc" is italicized to reflect its semi-variable character to be replaced by the accurate URL of a particular mirror.  This also emphasizes the specialized conceptual character of the term.   It is intended as an unobtrusive practice.  It hasn't been consistently followed anywhere else, up to now.
