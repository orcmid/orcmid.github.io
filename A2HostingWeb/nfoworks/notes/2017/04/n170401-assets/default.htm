This asset cache provides backup versions of materials gathered as part of SCE development.  These materials are retained for backup as verifiable sources for citations, quotation, commentary, and analysis.  In general, the primary source location should always be cited, with these cached items only provided as secondary sources, if at all. 

Asset materials are viewable and downloadable from this folder only if permitted by their authors/publishers.  Other assets that must be retained as part of the provenance of SCE work but that are not redistributable are preserved in seclusion.

Secluded Documents

Seclusion applies when access and redistribution is not permissible or has been withheld for any reason.  The material has been preserved in an inaccessible folder.  Whenever seclusion is no longer required, the material will be moved to this public folder and made accessible.  There is also unpublished material and other work products that may be carried in seclusion as a form of preservation and in support of provenance of the work.

Document Formats

Formats of documents are MHT, PDF, TXT, and ZIP.  Detached digital signatures have ASC extensions.

  • ASC file extensions are used on digital-signature files that provide detached PGP signatures for the file of the same name absent the .asc extension.  These are text files and there is information in the file itself on how to verify the signature along with unsigned but informative statements concerning what the signature represents.
      
  • MHT: cached MHTML (multipart/related) pages are directly viewable in some browsers, including Internet Explorer versions up to IE8.  Recent versions (9-10+) of Internet Explorer will usually download the file instead as a security precaution.  The downloaded MHTML document can be virus-scanned and viewed locally using Internet Explorer.  This format captures a complete web page, including its images and other direct inclusions.
                
  • PDF: cached PDF documents can be open in browsers or downloaded for saving local copies.  First check the size of the PDF and decide whether to force download rather than allow in-browser opening.
       
  • TXT: cached TXT (UTF-8 text file) pages are directly viewable.  They can also be downloaded without difficulty.  Because of different conventions for end-of-line coding, it may be necessary to view the files with software that is generous in the white-space code sequences that are recognized and handled.
     
  • ZIP: cached archive containing multiple files packaged using the Zip file format.  These are often large downloads.   Check the size first.  The material can be examined and extracted using an archive utility.

Basic Materials
