This document cache provides bibliographic resources and other resources in support of the explanation of the ODF Interoperability Assessment Helix.

Secluded Documents

Material is viewable and downloadable from this page only if explicitly permitted by its author/publisher.  Other materials are secluded and, although cited here, are not downloadable here.

Seclusion applies when access and redistribution is not permissible or has been withheld for any reason.  The material has been preserved at an inaccessible d150601-seclusion/ folder.  Whenever seclusion is no longer required, the material will be moved to this folder and made accessible.  There is also unpublished material and other work products that may be carried in seclusion as a form of preservation and in support of provenance of the work, if ever required.

Document Formats

Formats of documents are PDF, TXT, and ZIP.  Detached digital signatures have ASC extensions.

  • ASC file extensions are used on digital-signature files that provide detached PGP signatures for the file of the same name absent the .asc extension.  These are text files and there is information in the file itself on how to verify the signature along with unsigned but informative statements concerning what the signature represents.
                
  • PDF: PDF documents can be open in browsers or downloaded for saving local copies.  First check the size of the PDF and decide whether to force download rather than allow in-browser opening.
       
  • TXT: TXT (UTF-8 text file) pages are directly viewable.  They can also be downloaded without difficulty.  Because of different conventions for end-of-line coding, it may be necessary to view the files with software that is generous in the white-space code sequences that are recognized and handled.
     
  • ZIP: archive containing multiple files packaged using the Zip file format.  These are often large downloads.   Check the size first.  The material can be examined and extracted using an archive utility.
