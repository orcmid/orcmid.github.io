The secluded resources consist of backup versions of material clipped as part of Document Container File Profile development in nfoWorks folio n130401.  These materials are secluded because redistribution permission is not available.  They are preserved strictly as verifiable sources for citations, quotation, commentary, and analysis work product in the absence of better freely-available sources.  In general, these entries must not be used, unless it happens that the material becomes public, with the entry then moved to the public cache.  The citations provided in the public cache should always be used. 

Formats of cached versions are PDF, MHT, and TXT:

  • PDF: cached PDF documents can be open in browsers or downloaded for saving local copies.  First check the size of the PDF and decide whether to force download rather than allow in-browser opening.
      
  • MHT: cached MHTML (multipart/related) pages are directly viewable in some browsers, including Internet Explorer versions up to IE8.  Recent versions (9-10+) of Internet Explorer will always download the file instead as a security precaution.  The downloaded MHTML document can be viewed locally using Internet Explorer.  This format captures a complete web page, including its images and other direct inclusions.
      
  • TXT: cached TXT (UTF-8 text file) pages are directly viewable.  They can also be downloaded without difficulty.  Because of different conventions for end-of-line coding, it may be necessary to view the files with software that is generous in the codings that are recognized and handled.
       
  • ZIP: cached Zip archive.  It is best to download the file and view its contents using a Zip utility.  On current Windows operating systems, the downloaded archive can be opened directly in the Windows Explorer, similar to opening a folder.
