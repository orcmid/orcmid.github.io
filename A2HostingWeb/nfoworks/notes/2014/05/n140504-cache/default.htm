This document cache provides backup versions of material supporting the development of the ODF 1.2 Schema Reference in nfoWorks folio n140504.  These materials are retained for backup as verifiable sources for citations, quotation, commentary, and analysis.  In general, the primary source location should always be cited, with these cache entries provided as secondary (cache) locations, if at all. 

Document Cache material is viewable and downloadable from this page only if explicitly permitted by its author/publisher.  Other materials are embargoed and, although cited here, are not downloadable here.

Cached Document Formats

Formats of cached versions are MHT, PDF, PRN, TXT, XSLX, and ZIP.  Detached digital signatures have ASC extensions.

  • ASC file extensions are used on digital-signature files that provide detached PGP signatures for the file of the same name absent the .asc extension.  These are text files and there is information in the file itself on how to verify the signature along with unsigned but informative statements concerning what the signature represents.
             
  • MHT: cached MHTML (multipart/related) pages are directly viewable in some browsers, including Internet Explorer versions up to IE8.  Recent versions (9-10+) of Internet Explorer will always download the file instead as a security precaution.  The downloaded MHTML document can be viewed locally using Internet Explorer.  This format captures a complete web page, including its images and other direct inclusions.
      
  • PDF: cached PDF documents can be open in browsers or downloaded for saving local copies.  First check the size of the PDF and decide whether to force download rather than allow in-browser opening.
       
  • PRN : a TXT variation produced by Microsoft Office Excel as Formatted Text with spaces (rather than tabs) for indented material and empty columns.  The format is for simple printing as text with fidelity to the spreadsheet organization.  These files can be viewed as TXT and are to be treated the same way.
      
  • TXT: cached TXT (UTF-8 text file) pages are directly viewable.  They can also be downloaded without difficulty.  Because of different conventions for end-of-line coding, it may be necessary to view the files with software that is generous in the codings that are recognized and handled.
     
  • XSLX: cached XSLX file carrying an OOXML spreadsheet document.  This document can be viewed in some browsers. It can be viewed in current implementations of Microsoft Office Excel and other applications that provide interoperable support for the format.  The document can also be viewed using Microsoft Office Web and OneDrive.
      
  • ZIP: cached archive containing multiple files packaged using the Zip file format.  These are often large downloads.   Check the size first.  The material can be examined and extracted using an archive utility.
