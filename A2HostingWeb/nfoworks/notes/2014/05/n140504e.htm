                          BASELINE ANNOTATED ODF 1.2 MANIFEST SCHEMA
                          ------------------------------------------

     This is a faithful rendition of the OpenDocument-v1.2-manifest-schema.rng file
     presented as an annotated text for reference and discussion.  The annotations are
     limited to the addition of line numbers, links to referenced external materials,
     and cross-references among to definitions in the schema.  Permalinks are set on the
     line numbers, <definition>s, and the <element>/<attribute> entries.  For an
     authoritative copy of the schema itself, the source location should be consulted.

     The latest version of this annotated schema and further details are provided at
     <http://nfoWorks.org/notes/2014/05/n140504d.htm>.  This baseline version is managed
     at <http://nfoWorks.org/notes/2014/05/n140504e.htm>.

     This HTML web page has an embedded digital signature that can be used to verify its
     authenticity and the absence of any alterations.  The public key of the signature
     has PGP fingerprint 04D0 4322 979B 84DE 1077 0334 F96E 89FF D456 628A.  The key is
     at <https://people.apache.org/keys/committer/orcmid.asc>, any PGP key service, and
     <https://keybase.io/orcmid>.  Procedures for verification and creation of this
     signed page are described at <http://nfoWorks.org/dev/2014/05/d140502g.htm>.
