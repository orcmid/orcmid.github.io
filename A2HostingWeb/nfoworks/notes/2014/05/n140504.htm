 ODF 1.2 Schema Reference

To facilitate analysis and specifications based on the OASIS ODF 1.2 Standard, annotated versions of the ODF 1.2 schemas are prepared. 

These schemas are accurate renditions of those from the ODF 1.2 specification.  However, they are edited to contain links for cross-references and anchors so that particular schema entries can be linked from other web content.

Annotation to form a linkable and navigable schema reference is a progressive activity.  The work is iterated along with work that relies on the schema reference for specific purposes.

-- Dennis E. Hamilton
Seattle, Washington
2014-05-08
