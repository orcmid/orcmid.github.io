1. Usage

Elements/attributes/values using this namespace should never be encountered in documents "in the wild."  Any encounter of this namespace binding should be regarded as foreign and given foreign treatment in OpenDocument Consumers including those complying with the RCT Repaired Change-Tracking Extension Profile.

This namespace is used in test documents for verifying the resiliency of existing implementations to the presence of foreign attributes in change marks and other tracked-change-related elements.  The goal is to verify that there is no disturbance to existing implementations and that implementations treat documents will ignore unrecognized foreign attributes. 

In the event this namespace is encountered, this is the resolvable documentation of the namespace origin and its restricted definition.  It is to be treated as an always-foreign namespace and given default behavior rather than being distinctly recognized.

For information about the activity in which this namespace is used for regression and resilience testing, see RCT Extension Down-Level Resilience (d140602).
