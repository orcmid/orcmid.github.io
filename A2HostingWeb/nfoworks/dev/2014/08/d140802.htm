  PCT Protected Change-Tracking Namespace

nfoWorks>dev>
2014>08>

 0.02 2017-06-14 20:22

The nfoWorks namespace identifier http://nfoworks.org/ns/odf/1.2/pct# is for nfoWorks-defined extensions to implement Protected Change-Tracking Extensions beyond the OASIS Standard ODF 1.2 specification.

The namespace is used with ODF 1.2 foreign elements, attributes and attribute values that are "implementation-defined" for ODF 1.2 OpenDocument Extended Documents that are components of Protected Change-Tracking Documents.  The additional XML Documents introduced as part of PCT OpenDocument Packages also rely on the PCT Namespace (path). 

The primary description for the namespace is always available at ns/odf/1.2/pct#.
