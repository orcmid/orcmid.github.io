1. Introduction

1.1 The namespace {http://nfoworks.org/ns/example#} is defined to illustrate some provisions of nfoWorks URI-based Namespace names and specialized Identifiers.  This namespace URI is, by nfoWorks convention, resolvable to a web page, <http://nfoworks.org/ns/example/>. 

1.2 The ns/example# namespace is provided to illustrate the ways that nfoWorks namespaces, local names, and other URI-based identifiers are defined using web-page locations. 

1.3 Although there are different forms of the URI that will resolve to the same page, the located web page defines only one specific form as the Namespace name.  Other forms are not defined to be Namespace names. 

1.4 Similarly, all local names and similar identifiers resolve to explicit locations on web pages.  That treatment is illustrated here with the local name {http://nfoworks.org/ns/example#} myTerm.  The definition of this term is located at <http://nfoworks.org/ns/example#myTerm>.  This is the only defined Local Name.

1.5 The definition for local name myTerm of the ns/example# namespace also establishes the concatenated form <http://nfoworks.org/ns/example#myTerm> as an nfoWorks identifier.   That is by virtue of the definition.  It is not an automatic situation.

1.6 Note: technically, the definition for the ns/example# Local Name myTerm is at the specific location <http://nfoworks.org/ns/example/default.htm#myTerm>.  The abbreviated "official" URI takes advantage of how the web site delivers default pages for folders.  This provides abbreviated URIs and additional technical flexibility in how the web site is developed and operated.
