1. Using the Grant

1.1 This grant applies to all contributions made by its author to the LibreOffice project of The Document Foundation (TDF).

1.2 This is the only grant that the author has made with respect to LibreOffice contributions.  There is no different grant made to the LibreOffice project itself.

1.3 It is recommended that LibreOffice project releases that contain any contributions covered by this grant be used.  That is a way to ensure that maintenance and fixes subsequent to the contribution can be taken advantage of.  It is then necessary to comply with conditions of the LibreOffice license beyond those of this grant.

1.4 If the contribution is employed independent of the LibreOffice project to which it is contributed, it is prudent to keep records of where, when, and how the contribution was obtained, along with a copy of this grant.
