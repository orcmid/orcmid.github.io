1. Using the Grant

1.1 This grant applies to all contributions made by its author to projects of the Apache Software Foundation (ASF).

1.2 The author also has a specific contribution license agreement with the ASF.  This grant does not amend that agreement.  Instead, it makes the contributions available to others beside projects of the ASF.  To be covered by this grant, it is only necessary to obtain the contributions and, if necessary in the future, be prepared to demonstrate that they are the contributions by the author of this grant.

1.3 It is recommended that Apache project releases that contain any contributions covered by this grant be used instead.  That is a way to ensure that maintenance and fixes subsequent to the contribution can be taken advantage of.  It is then necessary to comply with conditions of the Apache License beyond those of this grant.

1.4 If the contribution is employed independent of the Apache project to which it is contributed, it is prudent to keep records of where, when, and how the contribution was obtained, along with a copy of this grant.
