On 2005-01-30, in preparing for my final project proposal submission, domain names were leased for TROST and what is now called the TROST Pilot.

TROSTing.org

This domain (the site where this page is situated) is set aside for the dissertation project, open-source development, and on-going development activities that promote TROST and the application of TROSTing over an increasing range of situations and technology platforms.  It is very specifically tied to the SourceForge TROST project.

TROSTing.net

This domain is reserved for any federation scheme surrounding authentication, attestations, and so on.  There is no particular approach in mind.  It is probably not like a keyserver system (e.g., http://www.pgp.net), but it is reserved for something in that spirit.

TROSTing.com

This domain is reserved for organizational use, if that becomes appropriate or necessary. Although this is commonly a level at which either commercial products or commercial support is provided, there is no particular business model in mind at this point.  The domain name is basically reserved against anyone else employing it for a commercial purpose.

worthiness.org

This domain happened to be available and is a placeholder for e-mail forwarding (and e-mail related identity) trust@worthiness.org.  Likewise, trust.worthiness.org can be an usable domain for the topic.  Other worthiness's can be added.  No use has been made of this domain so far.
