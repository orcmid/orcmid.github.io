[Loose Notes:]

This is the bootstrap version of the Project Folio Bootstrap Template.  That's what being version 0.00 signifies.  This page corresponds to the project folio front cover built with the template.  There are a number of other standard pages that are also part of this template.

What will appear here as the bootstrap continues is instruction on customizing this page and the other pages of the template to form an initial project folio in ready-to-use bootstrap condition.

This is the bootstrap scaffolding for the template itself.  It will become the material that is

  1. moved to the target location for a new project folio,

  2. there transformed into the bootstrap version of a new project folio,

  3. and then morphed further to carry the content of the new project folio.

There is a different folio that describes the history of this one and how to use the current one.  (It can't be here because it isn't part of the template, it's the accountable progression about the template.  That will be clear from examining the worked examples and usage here on TROSTing.org.)

The way this template is going to be moved from its own bootstrap seed to something more useful is as follows:

  • The template is going to be cloned to make a project folio in exactly the way this template is meant to be used.

  • As the clone is morphed to be the bootstrap of the intended project folio, these pages are updated to incorporate instructions for accomplishing that same thing in making other project folios.

  • Naturally, the ideal case is to use it to obtain that different folio that, when developed, will describe the project history of bringing this one to some usable version level.
