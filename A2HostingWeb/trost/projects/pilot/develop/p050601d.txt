2. TOOLS REQUIRED

   To access and use the material in this archive, the following resources
   are required:

       Internet Connection and Browser
                      for access to the material and download of the
                      P050601c.zip file.

       Zip-format archive access utility
                      The archive was produced with WinZip 8.1 SR-1 (5266)

       Utility for viewing or editing simple text files
                      Windows Notepad is sufficient.  No tabs are used
                      and word wrap should not be used in viewing. The
                      lines of text in the body are not wider than 80
                      characters.

       Web Page editor for reusing the template pages and applying the
                      the templates.  These pages were created using 
                      Microsoft Office FrontPage 2003 (11.6356.6408) SP1.
                      Some features of the pages are specific to FrontPage.
                      These are features that must be customized for use
                      on a different site regardless.

       Browser or utility for viewing Microsoft HTML (MHTML) pages carried in
                      .mht files. These files are in the format of SMTP e-mail
                      pages carrying multi-part/related content that can be
                      reassembled to present a web page as actually viewed.

       Additional authoring and development tools (such as version-control
                      software) for incorporating the templates in the 
                      development of a web site.
 

3. INSTALLING THE MATERIAL FOR USE

   The material can be extracted and employed as a model for making new
   templates that apply some of the principles also embodied in these.

   Because of the variety of uses that may be made, there are no specific
   instructions.


4. USING THE MATERIAL

   The only documentation of this material is incorporated in the material 
   itself.  There are tacit assumptions and undocumented dependencies
   reflected in this material. These remain to be captured and documented.

   The fundamental purpose of the present packaging is to provide a permanent
   record of the template version placed in use on 2005-06-23.


5. OTHER MODIFICATIONS TO THE MATERIAL

   The latest version of the 0.10 templates on the TROSTing.org site may
   change as the result of Included Page content.  This applies to the site
   logo in the upper-left corner and the Creative COmmons information at the
   at the bottom of each page.
