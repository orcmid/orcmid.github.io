3. INSTALLING THE TEMPLATE FOR USE

   The simplest way to Install the template is to open it in Microsoft Word.
   Word will open the file as a document for editing.  It can be reviewed,
   edited, and printed as an ordinary document.

   Double-click on the template file to launch it in Word.  Word will treat
   this as a template for making a new document.  

   To install the template, with or without your own customizations, use the
   Microsoft Word File | Save As ... menu selection.

   In the Save As dialog box, change the "Save As type:" selection to 
   Document Template (*.dot).  Word will change to the directory it uses for
   your personal templates.  Adjust the name of the file to the one you
   prefer for it.  "TROST-Report.dot" is recommended if you are using it
   to make TROST-project materials.

   When the template is stored successfully, close Microsoft Word.


4. USING THE TEMPLATE

   As you may have noticed during installation of the template, you can 
   simply double-click on a template and Microsoft Word will open a fresh
   document with that template as its initial content.  

   Once you've used Word to save the document as a template, Word will also
   find it automatically when

    - you open an existing document that depends on that template
    - you select File | New ... to create a new document

   In the New Document case some versions of Word will display a template
   directory with the normal choices offered, including the TROST-Report
   template that you installed.  

   In Microsoft Office 2003 Word, the default blank document template is 
   used automatically.  A "New Document" sidebar panel is also opened.  The
   sidebar provides sources of alternative templates, including ones you
   have used recently.  If you don't see the just-installed template in the
   sidebar, select the "On my computer..." link.  You'll see the new template,
   among others, on the General tab of the Template directory that opens.

   However you select the template, its use creates a fresh (unnamed) document
   with all of the template content as its initial material.  The TROST-Report
   template provides comments (in blue {...}-ing) for you to review and remove
   as you customize the template-provided material for your use.
