The catalog of any dependencies of this physical TROSTing.org mirror on its anchor site. 

Every nfoCentrale site is able to operate underneath an "anchor site."  The anchor site is a physical hosting or mirror location. It might be the root directory of a CD-ROM. It might be the IIS 5.1 development site on compagno, a development laptop/server.  It might be the physical hosting site at nfoCentrale.com.   References to site http://TROSTing.org are mapped to the proper location within a web-hosting anchor site.  

The web pages for TROSTing.org are developed in a way where there is no dependency on any specific anchor site.  There cannot be relative references above a given site either.   There is not assurance that such a reference will operate properly.

The following are the explicit dependencies on the existence of an anchor site that must be cured:
