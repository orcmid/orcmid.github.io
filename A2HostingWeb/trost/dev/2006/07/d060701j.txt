2. TOOLS REQUIRED

   To access and use the material in this archive, the following resources
   are required:

       Internet Connection and Browser
                      for access to the material and download of the
                      d060701f.zip file.

       Zip-format archive access utility
                      The archive was produced with WinZip 9.0 SR-1 (6224)

       Utility for viewing or editing simple text files
                      Windows Notepad is sufficient.  No tabs are used
                      and word wrap should not be used in viewing. The
                      lines of text in the files are not wider than 80
                      characters (with the possible exception of a single
                      version-control label at the end of each file).

       A C/C++ compiler for verifying the successful use of the include
                      file.  The two files have been tested together 
                      using a Visual C++ 2005 Express Edition project.


3. INSTALLING THE MATERIAL FOR USE

   The material should be unzipped into a directory of your choice for
   examination and trial compilation using your preferred C/C++ compiler
   system.


4. USING THE MATERIAL

   This is preliminary material that takes the first step toward a more-
   complete solution.  The progression is intended to demonstrate a trust-
   worthy approach to making portable interfaces to platform-specific
   implementations.  This version simply confirms that the header file
   is acceptable to C/C++ compilers.
