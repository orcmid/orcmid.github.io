{This page is a template for making the Diary & Job Jar for a new FAQtip folio.  Instructions of this form will provide details on the systematic customization to make before placing the page in use.}

{NOTES:

  • This page can be omitted if no diary & job jar is needed at this time.  In that case, it is often wise to reserve the a-page for future use as a diary page, if ever needed. 

  • If there is no a-page, the reference to one in the revision history of the folio cover page must be removed. }

{First Steps:
 
  1. The heading title of the page and the title banner, above, are customized in the same manner as the cover page for the target folio.
  2. In the title block, above, the first line can be replaced by copying from the already-customized folio cover page. 
  3. The second title lines should be replaced by an appropriate substitute based on the folio title.  The <head> title for this page should be updated accordingly (using File | Properties in FrontPage, for example).
  4. Similarly, the updated anchor-block path in the upper right can be replaced by copying the already-updated version from the folio cover page.
  5. The anchor for this page itself must have the link text changed from i000006a and the link should be corrected to refer to the page being built.
  6. Provide customization of other places where the {Customize} marker appears.  Delete markers (and other {...} text groups) when they have been acted upon.  When these top annotations are all deleted, the top of the table should provide the border beneath the title banner at the top of the page.
  7. Don't forget to adjust the revision history and the construction-history block as specified at the bottom of this page.}
