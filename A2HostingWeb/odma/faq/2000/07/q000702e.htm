4. Confirming Log Failure

Before concluding that the application or ODMA are failing to produce a log, there are a number of things to try.

Making sure the proper ODMA Connection Manager is installed.

Making sure the log settings are appropriate.

Making sure that the application is able to find the ODMA Connection Manager.

Making sure there is a default DMS or a specific DMS for the application

Making sure that the application is ODMA-enabled, as required (MS Office as an example).
