1. What is ODMA Tech?

The e-mail address ODMATECH@ftplist.aiim.org is the address of the ODMA Tech e-mail discussion list.

ODMA Tech is the general list for all broadcast to the community of people and organizations interested in the Open Document Management API (ODMA), its support, troubleshooting, compliant products, and further development.

4. Subscribing to ODMA Tech

To subscribe for receiving ODMATECH messages,

  • Using the e-mail service that you want to have receive messages from the list, create a simple text mail note
  • Have your message be the single-line request

    SUBSCRIBE ODMATECH

  • mailto:LISTSERV@ftplist.aiim.org
  • The list server will respond with confirmation and further instructions on using the list

5. Signing off ODMA Tech

To unsubscribe and remove yourself from ODMATECH,

  • In the same way that you subscribed, send the single-line request

    SIGNOFF ODMATECH

  • The list server will respond with confirmation information.

6. Other List Operations

For other information on using the lists, and customizing for your preferences, send message with the following commands in the message body

  • HELP

    provides basic information about the list service

  • INFO REFCARD

    provides a "reference card" of the operations you can perform with the list service.  The reference card is provided by return e-mail.

  • INDEX list

    returns an index to messages in the archive for list.
