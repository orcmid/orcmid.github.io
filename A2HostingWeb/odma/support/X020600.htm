1. Incident Summary (2002-06-06):

There are intermittent situations where an user of PowerPoint 97 will be unsuccessful in performing a Save to a DMS.

  1. The inappropriate error message, NODMS, is presented by the ODMA Connection Manager.
  2. When the user accepts the error message, PowerPoint simply reverts to where it was when the File | Save or File | Save As ... operation was requested.
  3. Repetition of the Save or Save As ... request leads to the same pattern.
  4. PowerPoint must be exited and restarted to break out of this condition.

1.1 Resolution

Under Investigation

1.2 Work Around

None known at this point

1.3 Configurations Confirmed

The problem has been confirmed with the following configurations:

PowerPoint 97 ODMA 2.0.0-1 ODMA32.dll Connection Manager Documentum Client
