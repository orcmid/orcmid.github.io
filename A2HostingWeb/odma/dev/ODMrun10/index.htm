You've reached the scaffolding for the ODMdev ODMrun10 section. 

This folder holds the source code and other software elements for implementation of ODMrun1.0, the ODMA 1.0 Test Runner.  ODMrun1.0 is developed in parallel with ODMref1.0,

  • As a progressively-developed test fixture that is used in test-first development of ODMref 1.0 functions and installation
  • As a fixture for creating complete logs of requests made during exercise of ODMA DMS integrations, including ODMref 1.0
  • As an appliance for capturing and replaying scripts for use in trouble-shooting, regression testing, and demonstration of identified defects in ODMA DMS integrations and the ODMA Connection Manager.
