  • Latest version: The latest ODMJNI 1.0 Deployment material can be found via   <http://ODMA.info/dev/devNotes/2006/12/d061201b.htm>.
  • Previous version:  ODMJNI 1.0 0.40alpha Document-Update Features <http://ODMA.info/dev/devNotes/2006/12/d061201f.htm>.   This release extends end-to-end document-access to allow for modification of accessed documents.  The modifications are used to update the managed version, with any change to the properties of the managed version also made available for presentation to the Java application.
  • Next version ODMJNI 1.0 0.60beta Public-Beta Release (under development)
    <http://ODMA.info/dev/devNotes/2006/12/d061201h.htm>.  This release consists of incremental updates until the full 0.60beta cumulative update is issued.  The incremental updates are made using this 0.50beta release as a base.
             
  • This page:  0.04 ODMJNI 1.0 0.50beta Feature-Complete Release <http://ODMA.info/dev/devNotes/2006/12/d061201g.htm>.   This completes the essential feature set for adoption and usage of ODMJNI 1.0 in beta integrations with Java applications.  There are places where testing is weak and where features are not at their stable behavior.  The API is stable and the features for conducting trial integrations are all available in usable form.
