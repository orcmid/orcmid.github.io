The info.odma.practical100 package provides basic interfaces, classes, and
other definitions for implementing ODMA 1.0 access from Java-based
desktop applications on Microsoft Windows.  The component interfaces are
designed to shield Java-based applications from ODMA API details.  The
package provides a practical level of ODMA-aware operations suitable for
typical office-desktop applications.

This distribution of info.odma.practical100 components includes deprecated
elements from earlier stages (from 0.30 to 0.58) and the refactored and
added elements that will be perpetuated in release 0.60 and beyond.  The
purpose of the release is to provide a transition to 0.60 for those who have
developed using earlier versions.

If you have not relied on earlier releases of ODMJNI 1.0, it is recommended
that deprecated elements in this package not be used.  It is even better to
wait for the release of the 0.60 Public Beta.

1. PREREQUISITES

   To make use of this distribution, it is necessary to have already
   satisfied the following minimum prerequisites:

      * Extraction of Zip material into subdirectories of folders on the
        Microsoft Windows platform (Windows 2000 and later)

      * Execution of console (command-line) sessions, navigation among
        directories, and executing Windows console applications and
        batch scripts.

      * Customization of batch scripts to correctly work with installed
        Java Standard Edition (J2SE) SDK software.

      * Creation of a development tree, starting at some folder, <folder>,
        in which is installed the practical100 development tree
        <folder>\info\odma\practical100 with subordinate test folders.


2. INSTALLATION

   This archive is designed to be extracted to the <folder> location that is
   the Java classpath point for the ODMJNI 1.0 packages on your computer.

   If there is already a <folder>\info\odma\practical100 path, the
   practical100 folder and its subfolders should be deleted after saving
   any customized OdmJava.bat script from that folder.

   Following extraction of the 0.59 practical100 material, any OdmJava.bat
   customization should be restored at

        <folder>\info\odma\practical100\OdmJava.bat
        <folder>\info\odma\practical100\test\Null01\OdmJava.bat

   There is also an OdmJava.bat within the regression set,

        <folder>\info\odma\practical100\0.30-practical100.zip

   To extract the material, place the archive,

        d061001h-practical100-0.59beta.zip

   in the <folder> directory.  Unzip the archive with folders preserved.

   The d061001h-ReleaseNotes.txt file provides further information on the
   nature of the changes introduced to practical100 for 0.59beta operation.
