1. Summary

This variation of the 0.05alpha material was produced to harmonize with the ODMJNI 1.0 0.20alpha release. 

The practical100\test\Null00 folder confirms that no regression has been introduced since the previous package.  This will also be the basis for regression tests of evolutionary increments beyond 0.20alpha.

2. Developer Notes: Changes for 0.20alpha

The following changes were made as part of developing this version:

  • The MyJava.bat script is replaced by the OdmJava.bat script also used with odmjni100 development. The script is cleaner and has more checks on correct modification.
      
  • The BuildAlpha.bat script is replaced by a BuildClasses.bat based on the version for odmjni100.  This provides a much simpler style without so much fancy business.
      
  • The practical100\test\Null00 folder is created and the original test is replicated there as Null00.java, with construction and testing using BuildNull00.bat and RunNull00.bat scripts.  These are in the simpler style used with odmjni100\test\Setup02.
