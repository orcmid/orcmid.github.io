This distribution of info.odma.odmjni100 components includes deprecated
elements from earlier stages (from 0.54 to 0.58) and the refactored and
additional elements that will be perpetuated in release 0.60 and beyond.
The purpose of the release is to provide a transition to 0.60 for those
who have developed using earlier releases.

If you have not relied on earlier releases of ODMJNI 1.0, it is recommended
that deprecated elements in this package not be used.  This should allow
operation as a preview of the release for 0.60beta.

1. PREREQUISITES

   To make use of this distribution, it is necessary to have already
   satisfied the following minimum prerequisites:

      * Extraction of Zip material into subdirectories of folders on the
        Microsoft Windows platform (Windows 2000 and later)

      * Execution of console (command-line) sessions, navigation among
        directories, and executing Windows console applications and
        batch scripts.

      * Customization of batch scripts to correctly work with installed
        Java Standard Edition (J2SE) SDK software.

      * Creation of a development tree, starting at some folder, <folder>,
        in which are installed the odmjni100 development tree
        <folder>\info\odma\odmjni100 with subordinate test folders.

   To work with and modify the JNI library, odmjni100.dll, it is also
   necessary to work with Visual C++ 2005 Express Edition and be able to
   make batch file customizations and command-line builds as part of
   creating odmjni100.dll implementations.

   THIS IS A COMPLETE REPLACEMENT.  PREVIOUS BETA DISTRIBUTIONS SHOULD BE
   REMOVED FROM THE odmjni100 BRANCH OF THE DEVELOPMENT TREE.  If there are
   locally-customized versions of batch files and other changes, they should
   be preserved before extracting the material in this distribution.

   To access and use the material in this archive, the following resources
   are required:

       Internet Connection and Browser
                      for access to the material and download of the
                      d061101o-odmjni100-0.59beta.zip file.  Microsoft
                      Internet Explorer 7 has been used to confirm
                      download from the web on the http://ODMA.info/ site.

       Zip-format archive access utility
                      The archive was produced with WinZip 9.0 SR-1 (6224)

       Utility for viewing or editing simple text files
                      Windows Notepad is sufficient.  No tabs are used
                      and word wrap should not be used in viewing. Most
                      lines of text in the body are not wider than 80
                      characters.  Some command lines in logs are wider.
                      The text files in this archive are all in UTF-8
                      encoding and were produced using jEdit 4.2final.

       A Java SE JDK such as the JDK 1.5 used to develop all of the Java
                      applications in this distribution.

       A Visual C++ 2005 Express Edition (or compatible release) plus the
                      Microsoft Platform SDK if it is desired to reconstruct
                      the odmjni100.dll.  The OdmVC++.bat and OdmPSDK.bat
                      files in the odmjni100 directory provide further
                      information on the configuration used.


4. INSTALLATION AND USAGE

   The entire archive should be extracted into a folder that can be used
   for review of all of the material.  A folder that is chosen for extracts
   of all ODMJNI 1.0 distributions is satisfactory.

   ** NOTE: If the same <folder> has been used to hold earlier materials
   from odmjni100 development, DELETE THE PREVIOUS MATERIAL AT PATH
   <folder>\info\odma\odmjni100\ and below.

   The <folder>\info\odma\odmjni100\ provides the sources used to develop
   the latest versions of the tests.

   To use the current set of regression tests, not necessarily from the
   same sources, the test archives at <folder>\info\odma\odmjni100\ should
   be extracted in that directory to provide the different regression sets:

       d071001e-0.54-odmjni100.zip extracts to 0.54-odmjni100\ for Clicker
       d071001f-0.57-odmjni100.zip extracts to 0.57-odmjni100\ for Check04
       d071001f-0.59-odmjni100.zip extracts to 0.59-odmjni100\ for Check04

   The 0.57 Check04 uses deprecated features and no new features.  The 0.59
   Check04 does not use deprecated features and relies only on new features.
   Each regression set has its own manifest with information on using the
   test and reviewing the results of tests already performed.
