Setup02 is used to confirm successful creation of an odmjni100.dll that uses the OdmNative classes and interfaces for its access to the ODMA API on behalf of a Java-based application.

The folder tree of Setup02 is structured for installation alongside the source trees for the other portions of ODMJNI 1.0 development: practical100 and OdmNative100.
