ODMJNI 1.0 is a Java-ODMA Integration Kit.  It provides a set of Java interfaces and classes for use in creating Java-based ODMA-aware Microsoft Windows desktop applications.

ODMJNI 1.0 development is in the final stages leading up to the 0.60 first Public Beta release.  The current 0.58beta and the intermediate 0.59beta are available as public previews.  The preview stages are used to transition to the interfaces and functions that are intended to be supported through to the 1.0 release.  The preview stages are also used to incrementally-evolve the site and materials to 0.60beta.

The 0.60 Public Beta release will be adequate for trial use in production applications.  Although future maintenance and stabilization releases are expected in the progression to a final 1.00 version, the interfaces and functions are expected to be stable.

Details of the current beta-level release are always provided at the ODMJNI 1.0 Public Beta Release page.
