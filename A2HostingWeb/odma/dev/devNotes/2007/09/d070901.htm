ODMJNI 1.0 is a Java-ODMA Integration Kit.  It provides a set of Java interfaces and classes for use in creating Java-based ODMA-aware Microsoft Windows desktop applications.

ODMJNI 1.0 Packaging is the portion of the project that is concerned with packaging of the ODMJNI 1.0 elements for deployment with applications, for use by application developers, and for use by ODMJNI 1.0 developers, maintainers, and customizers.

Packaging is the fourth leg on the ODMJNI 1.0 development stool.  Careful balancing ensures that the implementation of packaging is not wobbly.
