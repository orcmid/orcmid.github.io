This regression set is applicable to odmjni100 0.57 through odmjni100 0.59

It relies on elements that are deprecated in practical100 and odmjni100 0.59 and it will not be applicable to odmjni100 0.60 and beyond.

 

4. Developer Notes

2007-11-29-21:22 Surprise Failure Mode Test
While repeating a run of the regression set to confirm that everything works properly with a revised RunCheck.bat, I ran into an interesting failure mode.  Apparently, ODMASAMP has a limit on the number of documents it can deal with.  Consequently, any action that would give rise to a new document failed with an error message from ODMASAMP as well.  Although this demonstrates how badly limited ODMASAMP is as a reference and a confirmation check, it did provide failure modes that have not been tested.  The capture test transcripts and logs show the error messages and how the different ODMJNI 1.0 0.59 functions handle the exceptions that result.
