2008-02-15-20:29 Definition of Success
In addition to the external purposes to be satisfied by the ODMA32 2.5 Integration, the measure of success is that there are sufficient surrounding fixtures that it is possible to perform extensive regression testing of any ODMA Connection Manager using the ODMA32 2.5 fixtures.  It is only at this point that a maintenance release of the ODMA32 2.0 Connection Manager can be considered.
