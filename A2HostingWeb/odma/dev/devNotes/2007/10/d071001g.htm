{author note: This folio page is to quickly provide a receptacle for the regression set created as part of practical100 0.59 beta development.  The details will be filled in off the critical path.}

This regression set does not rely on any elements that are deprecated in practical100 and odmjni100 0.59.

It is applicable to odmjni100 0.59; it is expected to be applicable to odmjni100 0.60 and beyond.

 

4. Developer Notes

2007-12-01-12:46 The Basic Model for Future Regression Sets
Although additional documentation on setting up and running these tests is required, this version provides the basic model for regression sets until we have better automation of testing.  I am going to revise the earlier 0.30-practical100, 0.54-odmjni100, and 0.57-odmjni100 packages to be at the same level.  The odmjni100 0.59 package can be completed without those changes because the regression sets and the individual package releases are now decoupled.
