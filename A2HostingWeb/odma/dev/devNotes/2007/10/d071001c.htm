4. Developer Notes

2007-11-30-17:24 Missing Failure Mode Tests
Without the ability to inject error responses through the ODMA Connection Manager (or mock the Connection Manager), there are many failure behaviors that we cannot confirm and also demonstrate.  This is a significant part of the gap.  This was reinforced by seeing ODMASAMP, the ODMA Sample DMS fail because it was too full.
