This folder holds the source code and other software elements for implementation of ODMref1.0, the ODMA 1.0 Reference Implementation.  ODMref1.0 demonstrates a minimalist implementation of an ODMA DMS integration that is fully functional:

  • As an annotated, worked demonstration of important questions that must be resolved in arriving at a minimum ODMA-compliant DMS integration
  • As a demonstration and test fixture for verifying and trouble-shooting the operation of other ODMA components
  • As a template for creation of experimental, prototype, and production-grade derivatives useful in ODMA integrations
  • As a baseline for evolution of reference implementations having the additional functionality specified for ODMA 1.5, ODMA 2.0, and special ActiveODMA integration cases defined in the future.

This construction structure page is part of the engineering, construction management, maintenance, and infrastructure for ODMdev.  You are welcome to explore this and other construction-information pages.   You'll need your hard hat and safety shoes.
