A server migration has introduced broken linkages as part of changing to a web host that uses case-sensitive file names.  As a consequence, there may be broken links and inaccessible material.  A project for systematically repairing the site is underway.

The most-heavily used areas are repaired first, usually starting with material that is being worked on or updated.

The Miser Project is a practical exploration of key concepts at the foundation of computer-science.  

The project is undertaken as independent scholarship.  The Web is the vehicle for project communication and cooperative developments.  Project topics are expanded as hypertexts of individual web pages, including diaries and notes that are the annotations and history of the project. 

All of the work is available for use and redistribution/publication under a simple open-source license.
