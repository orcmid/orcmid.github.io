FILES FOR THE VICKI HAMILTON POSTCARD ORDER
-------------------------------------------

These are the files on this CD.  They were all made using Windows 98.

FOT0222-22-2002-08-29D.tif
	A 300spi compressed TIFF file of the cover image for the Victoria Hamilton 
	postcard order.  This file has RGB color channels.

FOT0222-22-2002-08-29F.tif
	An *uncompressed* version of the same file as FOT0222-22-2002-08-29D.tif,
	also with RGB color channels.

FOT0222-22-2002-08-29-postal.doc
	Is a Word 2000 document that shows the postal side of the card, using 
	Arial (TrueType).

FOT0222-22-2002-08-29-postal.tif
	Is a small compressed binary TIFF that shows the postal side using a 
	Papyrus font.  This is just for an idea of what we were trying out.
	The Word 2000 document provides an acceptable layout, and we accept
	the use of standard fonts for the text part of the postal-side
	of the card.
