Construction Log

This images directory is defined as part of the typical web structure for this site.  Here are the  images used in the overall organization of Millennia Antica on the Web and used on every image of the site.  This directory has a construction-oriented index.htm page and shares the common construction organization of Millennia-Antica.

Change History

2002-08-04-18:09 Create Image Map for Millennia Antica Entrance (orcmid)
1, PC Paint Shop Pro 7 was used to change the millennia-antica-sign.gif image to the millennia-antica-sign-2002-08-04.gif version, with an image map created in millennia-antica-sign-2002-08-04.htm.  
2. The Millennia Antica default.htm page is updated to use the new image and the image map, with the image map manually adjusted for use of the reduced image and also for mapping the image properly.
   
2002-01-08-14:57 Create Millennia Antica Welcome Sign and Perpetuate Theme (orcmid)
1. A large text sign is created for the site entrance, and incorporated as an image so that the shadowed Papyrus font is preserved in all browsers.
2. All of the intermediate forms of material used to make the Millennia Antica images is imported here, so that there is version control (using Microsoft Visual Source Safe) and the site provides backup of the materials used in its creation.
 
2002-01-07-22:07 Introduce Background Texture (orcmid)
1. To get closer to Vicki's themes, I searched for background images that would be appropriate as a reflection of pottery or something consistent with the papyrus format Vicki favors.
2. I used the clip-art gallery and went on-line for more to add from the Microsoft Design Gallery.  I found four and have imported the last one as millennia-texture.gif.
3. I introduced a default deep red font to use.  It provides the brick/brown close to the logo coloring.
 
2002-01-01-23:21 Update Logo Again (orcmid)
1. Starting with Vicki's business card, in Visio, a new version was created.  To save it properly, the image was created by linking the Visio into a Word 2000 document and saving that as HTML.  The resulting .GIF had some opaque white, and that was cleaned up to provide clean transparency using Microsoft Image Composer 1.5. The resulting images is that now on the site.
 
2001-12-29-21:11 Update Millennia-Antica Logo (orcmid)
1. The lidded-pot .BMP image was converted to a Microsoft Image Composer (.MIC) file.  The lettering is in Comic Sans font.
2. The .MIC file was saved as a .GIF with transparent white.
3. Microsoft Photo Editor was used to resize the .GIF image for better fit as the logo.
4. There is still a problem about truncation of the image on the right.  I think I can fix this using Word, later, when we have reviewed all of this and we see what we would be happier with.
 
2001-12-29-19:35 Create Original Millennia-Antica /images Section (orcmid)
