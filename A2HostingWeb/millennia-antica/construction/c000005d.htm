In addition to providing descriptive information and documentation, the master version of this page is also a template for non-construction pages in standard sections of the site.  The template features depend on the use of Microsoft FrontPage for the development and maintenance of this material, and are only fully available when these web pages are accessed using Microsoft FrontPage.  

Informal Template Usage

  1. This page is can be used as a boilerplate for a full page based on the default style for non-construction pages of this site.  In Microsoft Front Page, make a copy of the page and then derive the new page by transformational editing:

    • Use File / Properties ... to change the title of the page

    • Change the title in the middle table block at the top of the page.

    • Change the "grounding block" in the right section of the table to provide the full context of the present page and any version number and time stamp.

    • Update the creation information in the right section of the page bottom banner.

    • Delete this text section and the box at the top of the page, replacing it with your own page content.

    • Introduce your own version history annotations, if desired.  Make sure the links in the history are as you want them to be.

  2. The title block -- the table at page top -- can be copied by itself to produce a title block with the correct styles and the automatic inclusion of the correct left-section image for this site.

  3. The construction block -- the table at page bottom -- can be copied by itself to produce a construction block with the correct styles,  But you will have to replace the Hard Hat image inclusion and the "You are navigating ..." text inclusion.  The right-side material will be updated automatically by Microsoft Visual Source Safe, but the "created ..." line must be updated manually.

FrontPage "Include Page" Components

Title Image Link: C000005A

This component creates the table-cell image by inclusion of page C000005A.htm.  The component provides the image associated with this site, a title expressing the name of the site, and a shortcut to the site itself at its official URL.

Construction Image Link: C000005B

 

This component creates the table-cell image by inclusion of C000005B.htm.  This version uses anchor-relative location of the image and it assumes that every section using this link has an index.htm page as its construction-structure scaffolding.  

 "You Are Here ..." text: C000005C

You are navigating Millennia Antica

 This component creates the table-cell text by inclusion of page C000005C.htm.  The component provides a recurring text that we want to have present in the construction block of standard pages. The illustrated link is anchor-relative and always works.  There is room in the table cell for additional lines, and other text can be used as part of special page decoration.

FrontPage Style Inclusion 

The FrontPage Style inclusion used for this and other non-construction Construction Section pages is in page C000005e.htm.  That page describes how the background and basic styles are set on that page and how they are used from other pages.
