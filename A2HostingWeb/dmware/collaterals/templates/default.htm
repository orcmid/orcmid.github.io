FrontPage 98 Templates

The FrontPage 98 Templates are designed for addition to the template sets of FrontPage 98.  They are used to create new pages from within the FrontPage 98 Editor and Explorer. 

Each template is defined using a folder that is installed directly into the Microsoft FrontPage template configuration.  The templates are not designed to work with any other Web site creation and editing product.  There is a description page provided as part of the template.  The description page contains instructions for installing, using, and customizing each template. 

DMAware.tem - DMAware Stock HTML Page

DMAware.tem provides the stock HTML page originated for creating DMAware Web pages.  It has not been updated for DMware, and it has fallen into disuse.
