What is DMware?

Sponsored by AIIM International, the DMware Document-Management Interoperability Exchange supports openly-contributed document-management software, development kits, documentation, metadata definitions, and tools for confirmation and troubleshooting. 

DMware contributions include work of the Document Management Alliance (DMA) and the Open Document Management API (ODMA) coalitions.  DMware makes the coalition specifications and software available to the public.  The materials are preserved for voluntary support and continuation using an open-source model.

There are three DMware "legacy" archives: 

  • DMA, the specification and middleware for interoperable document-management integration from document-management clients
  • ODMA, the specification and middleware for interoperable integration of desktop applications with document-management clients on the desktop.
  • Reference Model, the informally-developed reference model for Document Management Services Integration

Newer materials are focused on the convergence of document management technologies on the Web:

  • WebDAV, Web Document Authoring and Versioning applied to managed documents

How Does DMware Operate?

The DMware exchange accepts any open-source contributions that promote document-management and information-processing interoperability.  Contributions can consist of documents, tutorials, specifications, computer software, and metadata definitions.

The DMware Model guides development and operation of the clearinghouse.  The model addresses

  • Availability of DMware material under open-source licenses
  • Contributing new and updated material to DMware
  • Conducting open-source projects under DMware
  • Creation of the DMware Metadata Exchange

What DMware is Available?

DMware preserves the work of the Document Management Alliance (DMA) and the Open Document Management API (ODMA) coalitions:

  • DMA 1.0 specification, samples, middleware, and trial-use extensions
  • ODMA 2.0 specification, samples, and connection manager

All specifications and software are available for download over the Internet.
