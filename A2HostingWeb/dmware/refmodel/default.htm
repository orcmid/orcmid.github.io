The DM Reference Model

Integrated use of DM Standards for the Enterprise

The DM Reference Model is developed by an informal working group of participants in Enterprise Document Management System standards activities.   The work is supported by AIIM International and The Black Forest Group.

The DM Reference Model  provides a framework for identifying the role of Enterprise Document Management standards in end-to-end enterprise applications.  This folder is the home for materials that address the following:

  • what it is the DM Reference Model?
  • how does the DM Reference Model clarify cooperative application of DM standards in end-to-end enterprise solutions?
  • how does the DM Reference Model support integration for reuse and for interoperability?
  • how do document-management standards fit into the overall scheme of enterprise solutions integration?
  • connections to the relevant standards activities, requirements sources, and sponsoring organizations
  • sources of more information
