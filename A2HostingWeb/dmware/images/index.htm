This web page, DMware/images/index.htm , is part of the engineering, construction management, maintenance, and infrastructure for the DMware Development Site.  You are welcome to explore this and other construction-information pages.   You'll need your hard hat and safety shoes.

This section is not intended to be explored as an ordinary Web page.  The images folder contains resources that are used by other web pages of the DMware development site and are generally not accessible or visible directly.  There is no effort to preserve access to this material from anywhere but pages internal to the site.

It is a general practice to have access to every section of the DMware site be mediated by a web page.   For sections that are for the private use of other DMware web pages, such as this one, it is unreliable to make absolute links to any of the material.
