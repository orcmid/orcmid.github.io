WinCon Construction Kit

The WinCon Construction Kit is provided in a versioned Zip package (e.g., WinConKit-0.1.0.zip).  The Kit includes template files, instructions, and tests for creation of WinCon construction sets.  The construction sets employ Microsoft command-line development tools for building programs on and for Microsoft Windows desktop PCs.

Purpose
Having reusable templates for systematic integration of WinCon construction sets into command-line build and operation of C/C++ programs on Microsoft Windows
  
Principles
1. Kits and derived construction sets rely entirely on already-installed utilities and freely-available Microsoft development tools for command-line C/C++ compilation.
 2. Kits provide samples and means for confirming and demonstrating successful construction-set integrations having the same quality.
 3. Resulting construction sets can be used by novices and hobbyists to easily build and trouble-shoot a project's stable source distribution by simply running the WinCon construction-set batch file that is part of the distribution.
 4. WinCon construction kits and derived construction sets have high conceptual integrity and adaptability for custom purposes.
  
Priorities
 1. Demonstration with simple open-source C/C++ programs built and used from the Windows command-line.
 2. Progressive introduction of documentation and improved stable distributions.
 3. Introduction of test, confirmation, and trouble-shooting support.
 4. Tie-in with sufficient tutorial and technical information for use in self-study and cultivation of developer software-engineering skills.
 5. Expansion of scope through inclusion of deployment and use in continuous integration.
  
Plan
 1. Start with prototyping in application to the construction of simple command-line game programs made available as open-source distributions.
 2. Extract initial stable kit from confirmed construction sets and the pilot customizations used for those kits.
 3. Iterate progressive improvement of functionality along with demonstration of how updated kits can be reliably applied in the updatng of existing construction sets.
  
People
 1. WinCon construction kits are intended for use by developers who are familiar with the working of  produced construction sets and are comfortable with the prerequisites and instructions for producing new construction sets from the WinCon kit.
 2. Intended use of the construction kit versions is to produce construction sets that can be used by novice developers, students, and those who rely on them to advance their skills in this area.  It is not necessary to be able to use the kit that such construction sets are derived from.
