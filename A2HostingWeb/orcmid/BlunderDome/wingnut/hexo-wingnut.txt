                     HEXO-GENERATED STATIC BLOG DEPLOYMENT

                     -------------------------------------

                                  

    This file is maintained at nfoCentrale/spanner/source/wingnut.txt on the

    author's PC.  The latest version is automatically included when the static

    blog pages generated by Hexo to nfoCentrale/spanner/public/ are uploaded

    to the blog's location on the orcmid.com hosted site.

    

    See <http://orcmid.com/BlunderDome/wingnut/index.htm> for more on the

    construction structure that the web-published material is spliced into.  

    Also see <http://orcmid.com/BlunderDome/wingnut/about/> for more 

    information.

    

    NOTE: During experimental development, wingnut/default.htm has precedence

    and the blog served at <http://orcmid.com/BlunderDome/wingnut> is that

    produced by Movable Type and about to be converted to the latest legacy.

    URL <http://orcmid.com/BlunderDome/wingnut/index.html> provides access

    to the experimental uploads until default.htm is taken over for the Hexo-

    generated wingnut blog.

       Once the Hexo-generated blog is stable enough to take over the public

    blogging for Spanner Wingnut's Muddleware Lab, index.html will be copied

    over default.htm as part of generation and publishing.  At that point,

    the previously-produced Movable Type and Blogger blogs will remain as

    legacy archives.  This is the pattern for switching other nfoCentrale

    blogs to static pages generated using Hexo.

       In the (unlikely?) event that there is yet-another migration to a

    different blog-generation mechanism, we'll need a way to push this one

    down to legacy as well.  Because the pages are static, there should be

    a way.

    

MANIFEST

    

    These files are uploaded from the generated PC folder, spanner/public/.

    Other files at <http://orcmid.com/BlunderDome/wingnut/> are produced by

    other means and co-exist alongside the Hexo-generated static blog material.

    

         wingnut.txt  This synopsis and manifest on spanner/public and its

                      presence in the http://orcmid.com/BlunderDome/wingnut/

                      blog folder.

        

             css/     cascading style sheets used in rendering the blog

             

        fancybox/     support for presentation via overlays of text, images,

                      videos, and slide-shows.  Version 2.1.5, although not

                      necessarily used.  <http://fancyapps.com/fancybox/>.  

                      

              js/     JavaScript support code

              

           index.html latest front page of the generated blog

           

           about/     The About page for the blog.  Applicable LICENSE.txt and

                      NOTICE.txt notifications are in this folder.  Preservation

                      of development materials is also housed beneath that

                      folder.

           

        archives/     archives for this current Hexo-generated blog

              

            2017/     Posts created in 2007.  Older yyyy/ folders are from

                      legacy blogs
