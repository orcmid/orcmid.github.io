1. This page serves as the  FrontPage Style Sheet for construction-structure pages of this site.  
        

  2. The purpose of the page is to be referenced in the <body stylesrc="c000003e.htm"> tag attribute of pages to be given the same style.
        

  3. The Style Source information obtained in this way provides the following information:

    • the color of the page background

    • any image/watermark used in that background

    • the default font color

    • the color of unvisited hypertext links

    • the color of active (being visited) hypertext links

    • the color of visited (including local) hypertext links
          

  4. In FrontPage 2000, the Style Source is set from the File / Properties / Background tab and specifying the "Get background information from another page" check box and page location.
        

  5. The contribution of this page can be reviewed by inspecting the HTML source or by viewing the page in Microsoft FrontPage.
        

  6. The use of the FrontPage Style Sheet introduces a custom stylesrc attribute into the <body> tag.  Although this tag is benign, it will provoke an error report from a checker for HTML 4.01 compatibility.
        

  7. Later versions of FrontPage do not provide this capability since it makes use of a special <body> tag attribute and HTTP styles can be used to accomplish the same thing.  However, the <body> tag can be edited to employ the attribute and it will be honored in versions as recent as FrontPage 2003.  We continue to use this authoring-time technique and avoid use of client-fetched style sheets or bulked-up HTML headers.
