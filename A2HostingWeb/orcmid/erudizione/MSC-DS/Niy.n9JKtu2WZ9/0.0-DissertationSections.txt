The material in this folder consists of a top-level master document and
nine subdocuments.  When this master document is worked on, all of the
files are in the directory 

    C:\M.Sc-IT-Liverpool\041128-Dissertation\MSC-DS-82-TROST\

This is draft 0.00, consisting mainly of outline, placeholder notes, 
and boilerplate for fleshing out into the dissertation chapter by 
chapter.

Using the Master Document
-------------------------

Generally, all files must be in the same directory for the linking of
subdocuments into the master document to function properly.

Start by opening the 0.0-FrontMaster.doc in Microsoft Word.  (Word 2003
is being used for creation, but the document should be usuable in 
any version starting with Word 2000.)

View the document in Outline format.  Enter Ctrl+\ to cause the sub-
documents to be included.  Switch to Print Layout view. The document
should now appear as one self-contained document having all
of the content from the subdocuments as well as the master.

** IMPORTANT NOTE: The master document introduces headings and chapter-
   relative page numbering that is useful for the creation of drafts.
   These styles will be corrected when the dissertation is produced as
   a single document for submission.
