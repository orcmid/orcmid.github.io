This folder provides Viaggio photographic images from roll F0508, begun around February 24, 2005 and processed on March 11:

  1. Visiting Cat.  This Askani-like cat came to the outside of my office window and was nosing around.  She has a definite black schmoo property here.
  2. Here she is aware of me and more alert to something happening over here.  Not so sharp, but fun.
  3. 2005-03-12 Blocco Setup Firmware Procedure.  Here is the first page that displays on arrival at the setup page when restoring it. This is the first-up screen.
  4. Initial Setup Screen.  Perhaps a little sharper.
  5. The default Startup Menu.
  6. The Startup Menu after Fastboot and Splash are unclicked and the version is brought up.
  7. Admin Password entry screen.
  8. Admin Password entry screen filled (probably the one to use).
  9. HDD Password menu option.  I haven't had the nerve to try this.
  10. USB Legacy Support is checked by default.  I don't know what it means to uncheck this.
  11. Boot sequence menu shows nothing set up to boot, and with FDD, CD-ROM, and HDD, in that order.  The individual sequence selections don't seem to do anything.
  12. Floopy [sic], CD-ROM, HardDisk, PXE [?] dialog.  The explanation is at the bottom, though I often miss it.
  13. Once the Admin Password is set, the Set User Password is enabled.  (Should be shown there.)  If no User Password is set, this BIOS will still ask for a boot password and the Admin Password is what it takes.
  14. New User Password entry screen.
  15. Save and Exit Screen
  16. Save and Exit with the Reboot confirmation dialog (OK to use this one).
  17. The Boot Password Screen that comes up when a password is set for Startup
  18. The Boot Password Screen with a password entered (use this).
  19. New User Password entry screen entered (use this one instead of #14)
  20. Default Boot Sequence Screen (again)
  21. Boot Sequence Rearranged the way I want it (HardDisk always first)
  22. Boot POST without the Fast Boot and Splash Screen (the logo is washed out)
  23. Better Boot POST with LAN Boot option shown too (logo is more visible)
  24. Another Boot Password dialog after the POST stuff (don't use this one, use the one with wrong password)
  25. The basic log-on screen, with my wounded pen in the side pocket.

Any derived images are not accessed by navigation to this folder.  They are reached by links that have been published for them in other pages.
