The construction set is everything needed to build, back-up, rebuild, and replicate the Libertà system.  This provides a basic lifecycle support for a typical Windows PC desktop system on a SOHO network, such as Centrale.

Summary

The Libertà Construction set is the collection of materials that are used to rebuild an operating Libertà configuration.  The basic principle is as follows:

  • The latest CD-ROM-distributed commercial software that is installed on Libertà is kept as part of the construction set.  This software is not backed up.  It will be reinstalled as necessary.  License information is preserved and backed up if necessary.

  • Additional commercial, free-ware, and downloaded software is kept in a construction section that is part of the Libertà hard-drive configuration and also backed up on a CD-ROM.  

    • This is where license information and also information on what to do to restore Libertà are retained.

    • This material is kept organized so that it always fits on one CD-ROM.

    • Configuration parameters and other information that are part of the fundamental configuration are also backed-up in the construction section.

    • Drivers and vendor/configuration-specific hardware-supporting software is also backed-up here if not available on separate CD-ROMs.

  • Other updates, such as those that are installed automatically as part of Microsoft Windows and the Microsoft Office System may be noted in the construction section, but they are backed up separately on CD-ROM if at all.

    • These updates are backed up if there is ever a need to be able to recover them after support ends for a Microsoft product.

    • The general setup process is to re-install all of the updates from the Microsoft site.

    • The re-installation process is designed to be as secure as possible, so that the system is unlikely to be vulnerable to attacks for which security updates are to be reinstalled.

  • Key data, such as Vicki's Outlook post-office and personal files are backed up on a separate CD-ROM.

The idea is that under ordinary conditions the only backup processes are for Vicki's personal files and for the construction CD-ROM:

  • These sets are regularly backed up to a directory section on a different Centrale computer.

  • These sets are recorded on a CD-ROM somewhat less frequently.
