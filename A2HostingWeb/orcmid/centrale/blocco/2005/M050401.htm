There is upgradeable firmware included as the ROM BIOS and Setup Configuration software.  This firmware may itself be revised, and it is certainly identified by version(s).  Each version has a default configuration.

The firmware also stores an user-specified configuration.  We need to keep track of those provisions as part ot configuration operation.

So there is a join (firmware version, configuration settings, dates of change) that goes into effect.  The default configuration is one of the configuration settings.

There is then some description about how the configurations work, what the settings mean etc.

If there are incidents involved, we link to them.
