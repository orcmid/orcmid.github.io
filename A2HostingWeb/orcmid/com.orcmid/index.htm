Folder Convention

Although the normal mapping of com.orcmid to file-system structures is via a com/orcmid/ directory tree, we use the abbreviated form here for two reasons:

  1. The structure is not intended to be included in a Java class path for direct use
  2. It is used only for publishing components under the com.orcmid package and sub-package names, making it unnecessary to waste a subdirectory level.

  • This is a web-site presence that provides a physical manifestation of Java package qualifier com.orcmid.
  • It collects packaged software under that qualifier
