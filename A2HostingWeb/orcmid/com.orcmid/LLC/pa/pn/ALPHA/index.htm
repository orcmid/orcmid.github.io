You've reached the ALPHA development section for illustration of Peano Numerals (pn) as part of a treatment of Peano Arithmetic in computational terms.  This is a careful software implementation that implements an abstraction of Peano numbers.  The abstract theory can be thought of as one source of requirements for the implementation(s):

  • The abstract perspective is explored in the Astraendo section of The Miser Project.  The abstraction and the bridge to a Java interface "contract" are discussed on the Numbering Peano blog.  There is an Astraendo Project that provides background and key materials for the discussions.
  • The realization of the abstractions in computer software using Java programs is explored here.  The elements of implementation and experiences with it are discussed on the Professor von Clueless in the Blunder Dome blog.  Sources of project materials for this domain of activity are identified below.

The ALPHA section is for material that is provisional, subject to change without notice, and is not yet incorporated in Java packages.  This material is sufficient for provisional usage and bootstrapping up to a packaged level, either a BETA package or a plane-versioned release.  (The use of capitals is to alert users of the material to their highly-provisional and experimental nature.)

Project materials provide additional information:

  • J040601: Peano Numerals Project for overall coordination of the software side
  • J040602: Peano Numerals ALPHA Organization for the organization of materials under ALPHA conditions
  • J040603: Compiling the ALPHA Peano Numerals where we confirm some things about how Java works to allow us to operate an Alpha level without creating too much rework in moving to production levels.

There will be further projects and materials introduced from time to time.  The overall Peano Numerals section and the projects there provide the essential roadmap and account for what is available.
