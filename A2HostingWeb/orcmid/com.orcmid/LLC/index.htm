You've reached the Orcmid's Lair Java packages section for Logic, Language, and Computation (LLC).  This section is for software projects that illustrate the interplay of logic, language, computation, and mathematics using concrete programs for the Java Language, the Java computation model, and Java platform implementations.

This section carries material designed to illustrate a number of aspects of 

  • syntax versus semantics versus behavior versus abstraction using programming languages
  • manifestation of abstractions using a very simple realization of Peano Arithmetic, namely Peano Numerals.
  • demonstration of the degree to which such a manifestation is dominated by platform-dependent notions and accidental characteristics necessitated by the platform

Subordinate Packages

  • pa - Peano Arithmetic, explorations in the rudiments of arithmetic from a logic and object-oriented-computation perspective
