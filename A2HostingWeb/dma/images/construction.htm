Construction Log

The images directory is defined as part of typical web structure also used by FrontPage extensions for this site.  For ODMA.info, this directory is preserved as part of standard site structure for independent mirroring.  The directory is used to carry images that are used at the AIIM DMware ODMA site level.  The directory has a construction-oriented index.htm page and shares the common construction organization of nfoCentrale.net sites.
