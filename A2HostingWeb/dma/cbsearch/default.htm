Content-Based Search Extensions to DMA 1.0

Last updated 2001-05-03-16:34 -0700 (pdt)

The Document Management Alliance (DMA) developed Content-Based Search Extensions to the DMA 1.0 Specification.

The DMA 1.0 Specification provides a model, integration framework and Application Programming Interfaces for use in creating interoperability among document-management software products, systems, services, and solutions.

The Content-Based Search (CBSearch) Extensions were developed by a working subcommittee.  This work is now in a form for introduction into trial-use and confirmation in practice.  No further development will occur under the auspices of DMA.

This area features the CBSearch trial-use extensions to DMA 1.0:

  • what is CBSearch,
  • how it fits with DMA 1.0,
  • how CBSearch works,
  • how CBSearch is used to provided interoperability among content-based search systems,
  • where CBSearch fits in the overall scheme of things,
  • where CBSearch is going,
  • where to find out more,
  • other resources
