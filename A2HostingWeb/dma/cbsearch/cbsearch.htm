DMA CBSearch

Content-Based Search Extensions to DMA 1.0

The Document Management Alliance (DMA) has developed Content-Based Search Extensions to the DMA 1.0 Specification.

The DMA 1.0 Specification provides a model, integration framework and Application Programming Interfaces for use in creating interoperability among document-management software products, systems, services, and solutions.

The Content-Based Search (CBSearch) Extensions were developed by a working subcommittee.  This work is now in a form for introduction into trial-use and confirmation in practice.  No further development will occur under the auspices of DMA.

This is Zip Package 0.02 of the Trial-Use CBSearch materials.  It is not the definitive source.  To confirm the availability of the latest definitive package of materials, check at the InfoNuovo CBSearch project web site for location of any more-authoritative version.

About this Material

Download Locations

To obtain the latest information , look in two places: ftp://aiim.org and http://www.infonuovo.com/dma/cbsearch/.

  • The AIIM DMA FTP Site's ftp://ftp.aiim.org/DMA/TechnicalCommittee/CBSearch/ subdirectory is the definitive location for DMA CBSearch information. 

    ftp.aiim.org is a moderated site.  You will need an User ID and a Password to access the material.  All members of DMA have access.   If you don't know the password, request it from one of the officers of the DMA Technical Committee.

    The official materials will be moved to a public location of the DMware Clearinghouse.  Until that is accomplished, http://www.infonuovo.com/dma/cbsearch is an alternative source.

  • The InfoNuovo.com site DMA Clearinghouse location http://www.infonuovo.com/dma/cbsearch is an alternative public location.   A recent version of the material is at that location in the form of viewable web pages.  This is provided as a convenience for DMA 1.0 users and adopters who find it difficult to access ftp.aiim.org.  Until the DMware Clearinghouse is operational, the definitive version is on ftp.aiim.org.
