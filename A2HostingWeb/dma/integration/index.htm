You've reached www.infonuovo.com/dma/integration , a collection of information about the DMA 1.0 Integration Model and its Extensions.

  • what the DMA integration model is,
  • how it fits on different platforms,
  • how it is used to provide interoperability among document management systems and services on the same and different platforms,
  • the benefits of the integration model for plug-and-play installation and reuse of portable implementations,
  • how the DMA integration model permits integration for interoperable harmony with other standards
  • sources of more information

This web page, www.infonuovo.com/dma/integration/index.htm, is part of the engineering, construction management, maintenance, and infrastructure of the DMware DMA collection.  You are welcome to explore this and other construction-information pages.   You'll need your hard hat and safety shoes.

If you'd rather find out about DMA's approach to Plug-and-Play integration and friendly platform independence than admire the scaffolding, organization, and implementation of the infonuovo.com site, go directly to the DMA/integration web folder.   If that doesn't work, come back later.  The folder is blocked off whenever the construction crew is working here.
