Welcome to the Document Management Alliance's DMA 1.0 specification!

You should unzip this zip file into a single directory.  You can use
any directory.  To view this HTML rendition, start your browser on
index.htm, the root of the HTML web.  From there, you can follow
hyperlinks to every section of the specification, including another
zip file in the Appendix containing the DMA 1.0 header files and
sample source code.

This draft of the specification has been approved by the DMA Advisory
Council for general use, as of December 24, 1997.
