Reference Middleware (dma10.dll) Version B0.03

  • Version B0.03 is the latest openly-distributed implementation of the DMA 1.0 System Manager.  It includes all source code.   There is simple test code for a trial System Object that can be used in development and in confirming correct installation of the System Manager.
