Orcmid Materials Are Being Repurposed/Preserved/Repaired on GitHub.

Migration to new servers introduces a number of problems that require repairs.  In addition to those defects, some older pages also need to be modified and reorganized for consistency across The orcmid.github.io organization.  While the site is being preserved to accomplish both kinds of improvements and adjustments, you may notice discrepancies and problems with links to site content.

 
 

You're at the entry level of the Orcmid on GitHub web site, https://orcmid.github.io.  This top level of site content is referred to as the front porch of the site.  All public content is reachable by entry through the front porch, whether by browser navigation through links or by use of a URL that specifies a path deeper into the site. 

This Construction Structure is the scaffolding on which the useful content and other web resources are organized.  Every accessible folder on the site can be found by navigation of the construction structure.  Some of them, such as this one, have a companion Construction Zone that is used to hold work items and details of that point of the construction structure and it subordinate construction structure.

This scaffolding and construction structure is all about the web presence and not about the authoring of those pages except when they are directly authored by FrontPage or similar software tooling.  The repository from which GitHub pages are produced might have authored forms, such as Markdown files.  From the perspective of the construction structure, it is the transformed-to-HTML pages that are accounted for.  Typical of this case are index.html files which here are generally produced from index.md files in the repository folder from which the GitHub Pages are produced..

This web page is part of the engineering, construction management, maintenance, and infrastructure of the Orcmid on GitHub web presence.  You are welcome to explore this and other construction-information pages.   You'll need your hard hat and safety shoes.

If you'd prefer reading about the Orcmid projects/repositories instead of admiring the scaffolding, materials, and construction equipment on the site, go directly to the Orcmid on GitHub entrance.   If that doesn't work, return later.  Projects can be blocked off whenever the construction crew is blasting in the area.
