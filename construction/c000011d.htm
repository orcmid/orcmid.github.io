Site Feed "All-Clear" Entry

When a testing period is ended, the following message is inserted in a new entry to announce the return to normal operation:


All Clear: End of Test
 
Any entries since the last " Caution: Feed Testing"
announcement were part of the test activities.
 
Consult the Site Status page for further information.
 

This will be inserted as the content of most-recent entry at the front.  Although it is possible to make this a solitary feed, it is generally preferable to add the entry into the existing feed and allow it to show up as the latest posting.

This message assumes that status for this site can be found as part of the combined status reported on Orcmid's Lair.

All-Clear Entry XML

This procedure is essentially identical to the "Caution: Feed Testing" announcement.

The file is named lair-atom.all-clear.xml.  It is stored in the same directory as lair-atom.xml so that it is easy to find for editing.

Before an exercise of the file, it needs to be prepared for the next usage.  This is accomplished by checking out the file 

  • Advancing the number in the <title> and the <id> of the entry.

  • Saving the modified file for check-in (thereby remembering the last-used number).

  • Pasting the entry portion into the lair-atom.xml file.

  • Updating the lair-atom.xml file's <modified> element and the <modified> and <issued> elements of the new entry.

  • Checking the lair-atom.xml file back in and updating the site.

The lair-atom.all-clear.xml includes <comment>s that provide instructions for its use.
