This page specifies some construction principles for ordinary pages of Orcmid on GitHub.  All sections start out with this form as a basic default structure, with more-decorated pages reserved for special purposes.  Although this is a Construction Material page, it is presented in Standard Page format for its usability as a template for such pages.

In addition to providing descriptive information and documentation, the master version of this page is also a template for standard, undecorated pages in sections of the site.  The template features depend on the use of Microsoft FrontPage for the development and maintenance of this material, and are only fully available when these web pages are accessed using Microsoft FrontPage.  

Informal Template Usage

  1. This page is can be used as a boilerplate for a full page based on the default style for ordinary, undecorated pages of this site.  In Microsoft Front Page, make a copy of the page and then derive the new page by transformational editing:

    • Use File / Properties ... to change the title of the page

    • Change the title in the middle table block at the top of the page.

    • Change the "grounding block" in the right section of the table to provide the full context of the present page and any version number and time stamp.

    • Update the creation information in the right section of the page bottom banner.

    • Delete this text section and the box at the top of the page, replacing it with your own page content.

    • Introduce your own version history annotations, if desired.  Make sure the links in the history are as you want them to be.

  2. The title block -- the table at page top -- can be copied by itself to produce a title block with the correct styles and the automatic inclusion of the correct left-section image for this site.

  3. The construction block -- the table at page bottom -- can be copied by itself to produce a construction block with the correct styles,  the automatic inclusion of the left-side Hard Hat image and link, and the "you are navigating ..." information in the center section of the block.  The right-side material will be updated automatically by Microsoft Visual Source Safe, but the "created ..." line must be updated manually.

FrontPage "Include Page" Components

Title Image Link: c000005b

This component creates the table-cell image by inclusion of page c000005b.htm.  The component provides the image associated with this site, a title expressing the name of the site, and a shortcut to the site itself at its official URL.  The image shown will be the one that was active at the date this page was last published.

 

 

 

Construction Image Link: Hard Hat Thumbnail

 

The image is inserted directly from the images/ folder. Add the image description "Hard Hat Area" and add the hyperlink to "index.html."  This almost always works for a given standard page where construction structure is present.

The use of the c000005c Include Page is not recommended because the link to nearby index.htm and Construction Structure is improperly relatived by FrontPage.

 

 "You Are Here ..." text: c000005d

You are navigating Orcmid on GitHub

 This component creates the table-cell text by inclusion of page c000005d.htm.  The component provides a recurring text that we want to have present in the construction block of standard pages. Because it is often the case that we want to put in a relative link, this block is not always used.  The stock link is anchor-relative.

FrontPage Style Inclusion 

The FrontPage Style inclusion used for this and other non-construction pages is in page c000005e.htm.  That page describes how the background and basic styles are set on that page and how they are used from other pages.  The link to this material must be set on the

<body stylesrc="c000005e.htm">

start tag of the page's code.
