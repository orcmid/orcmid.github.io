  • This page is the specification of construction principles for construction material itself, in the construction section and elsewhere.

  • It establishes standard c000000-c000005 pages for example..

  • It specifies how the construction team (e.g., orcmid) are represented here.

  • It specifies any special principles that apply in the construction material for cloaking accommodation and other purposes.

  • This page is a template for this and similar construction-material pages.

Note that the construction-material pages are different than the construction-structure pages.  The construction-structure applies to pages that are the skeleton beneath all sections of the site.  Construction-material pages are pages about the construction structure, the organization of construction information, and anything else to do with the construction of a site.

For orcmid.github.io, onstruction-material pages may appear in any section.   It is assumed that every section that does have construction material also has a construction.htm page (which is itself a construction material page).  Also, relative references are used so that these templates may be used from any place in the site, so long as the overall structural assumptions hold.

In addition to providing descriptive information and documentation, his page is also an exemplar for construction-material pages on this site.  The template features depend on the use of Microsoft FrontPage for their development and maintenance.  The features are only fully available when these web pages are accessed using Microsoft FrontPage.  

Informal Template Usage

  1. This page is can be used as a boilerplate for a full page based on the default style for construction-material pages of this site.  In Microsoft FrontPage, make a copy of the page and then derive the new page by transformational editing:

    • Use File / Properties ... to change the title of the page

    • Change the title in the middle table block at the top of the page.

    • Change the "grounding block" in the right section of the table to provide the full context of the present page and any version number and time stamp.

    • Update the creation information in the right section of the page bottom banner.

    • Delete this text section and the box at the top of the page, replacing it with your own page content.

    • Introduce your own version history annotations, if desired.  Make sure the links in the history are as you want them to be.

  2. The title block -- the table at page top -- can be copied by itself to produce a title block with the correct styles and the automatic inclusion of the correct left-section image for this site.

  3. The construction block -- the table at page bottom -- can be copied by itself to produce a construction block with the correct styles,  the automatic inclusion of the left-side Hard Hat image and link, and the "you are navigating ..." information in the center section of the block.  The right-side material will be updated automatically by Microsoft Visual Source Safe, but the "created ..." line must be updated manually.

FrontPage "Include Page" Components

These features are are incorporated in pages by using the FrontPage Insert / Component / Include Page ... menu.  The content of the include page will be incorporated where inserted.  In addition, any time the included page is updated, so will all places where it has been inserted.  When the pages are used outside of FrontPage, they behave as normal web pages with the last-included material shown.

Title Image Link: c000004b

This component creates the table-cell image by inclusion of page c000004b.htm.  The component provides the image associated with construction materials on thissite, a title expressing the nature of the page, and a relative link to the construction-material folder..  This inclusion works at any level of the site.

 

 

Construction Image Link: c000004c

This component creates the table-cell image by inclusion of page c000004c.htm.  The component provides the bottom-strip image to be associated with the nearby construction structure or construction zone, if any, along with the name of that feature (usually "Hard Hat Area") within which this construction material is contained.   All links are anchor relative, and the insertion can be used from any level of the site.  The immediate construction structure is meant to be the target.  It may be necessary to work this out manually other than with the include page, because of relative addressing issues.

FrontPage Style Inclusion 

The FrontPage Style inclusion used for this and other non-construction Construction Section pages is in page c000004e.htm.  That page describes how the background and basic styles are set on that page and how they are used from other pages.
