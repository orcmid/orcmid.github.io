Site "Caution: Feed Testing" Entry

When it is necessary to conduct a test or fire drill with a feed, a new entry is posted with the following message:


Caution: Feed Testing
 
This site feed is undergoing live tests as part of
Incident Response Procedures
for Orcmid's Lair and associated sites.
 
There will be an "All Clear" entry when testing ceases.
All entries prior to the next "All Clear" are part of the testing.
If there is no "All Clear" within 60 minutes of this announcement
it may be necessary to re-subscribe to this site feed.
Consult the Site Status page for the latest information.
 

This will be inserted as the most-recent entry at the front.  Although it is possible to make this a solitary feed, it is generally preferable to add the entry into the existing feed and allow it to show up as the latest posting.

This message assumes that status for this site can be found as part of the combined status reported on Orcmid's Lair.

Feed-Testing Entry XML

The file is named lair-atom.testing.xml.  It is stored in the same directory as as lair-atom.xml so that it is easy to find for editing and for usage.

Before an exercise of the file, it needs to be prepared for the next usage.  This is accomplished by checking out the file 

  • Advancing the number in the <title> and the <id> of the entry.

  • Saving the modified file and checking it in (thereby remembering the last-used number).

  • Pasting the entry portion into the lair-atom.xml file.

  • Updating the lair-atom.xml file's <modified> element and the <modified> and <issued> elements of the new entry.

  • Checking the lair-atom.xml file back in and updating the site.
